# Architecture Overview

## System Overview
AccessEdge Mobility is a dual-portal smart mobility system designed for accessibility-first transportation.

## Core Components
- User Portal
- Guardian Portal
- Backend API
- Database
- SOS Engine
- Routing Engine

## Data Flow
User → Frontend → Backend → Database  
                     ↓  
               Guardian Portal

## SOS Lifecycle
1. User triggers SOS
2. Backend logs emergency
3. Guardian notified instantly
4. Live tracking activated
5. Resolution status updated

## Future Scope
Digital cockpit integration using vehicle telematics.
